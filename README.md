# 🧮 Calculator App

A simple command-line calculator written in Python that supports basic arithmetic operations:

- Addition (+)
- Subtraction (-)
- Multiplication (\*)
- Division (/)

---

## 📌 Features

✅ Add two numbers  
✅ Subtract two numbers  
✅ Multiply two numbers  
✅ Divide two numbers  
✅ Support expressions like `2 + 5 * 10`
